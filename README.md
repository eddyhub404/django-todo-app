# Django TODO App

A basic CRUD TODO application built using Django and PostgreSQL.

## Live Demo

https://django-todo-app-ekum.onrender.com/

## Features

- Create Tasks
- Update Tasks
- Delete Tasks
- Toggle Complete/Incomplete
- Dynamic URL Routing
- Form Handling with POST Requests
- PostgreSQL Database Integration
- Responsive UI using Bootstrap

## Tech Stack

- Python
- Django
- PostgreSQL
- Bootstrap
- HTML/CSS
- Git & GitHub
